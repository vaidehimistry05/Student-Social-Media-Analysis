# 📊 Social Media Usage Analysis Among Students

## Overview

This project analyzes social media usage patterns among students using Python and various data analysis techniques.

The objective of this project is to understand how social media usage influences academic performance, mental health, sleep patterns, and addiction levels by extracting meaningful insights from a real-world dataset.

The project demonstrates the complete data analysis workflow, including data preprocessing, statistical analysis, exploratory data analysis (EDA), feature engineering, and data visualization.

---

## Features

- Data Loading
- Data Cleaning & Preprocessing
- Statistical Analysis
- Exploratory Data Analysis (EDA)
- Country-wise Analysis
- Risk Score Calculation
- Age Group Classification
- Usage Level Classification
- High-Risk Student Identification
- Ideal Student Recommendation
- Data Visualization
